# Usage

## Usage requirements  

### Usage Components

  1. Adaptive Switch x 2 ( Example: [AJelly Bean](https://www.ablenetinc.com/jelly-bean) )



## Usage Process

### How to Use?

  1. Press Power Button for 2 seconds to turn on the device.
  2. Press Adaptive Switch A or B to increment switch counts.
  3. Press Button A ( Largest button under the screen and it's labeled as M5 ) ro reset the attempt counts.
  4. Press Power Button for 6 seconds to turn off the device.
